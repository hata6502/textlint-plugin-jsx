> JSX and TSX support for textlint.

## Install

```sh
npm i textlint-plugin-jsx
```

## Usage

```json
{
  "plugins": {
    "jsx": true
  }
}
```

```json
{
  "plugins": {
    "jsx": {
      "extensions": [".custom-ext"]
    }
  }
}
```
